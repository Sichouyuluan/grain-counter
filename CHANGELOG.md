# 变更日志

## 2.0.0 (2026-05-11)
### 重构变更
- 将低带宽 Web 服务 (HTTP2.1) 重构为完整 FastAPI 应用
- 代码模块化拆分：graincounter/ 共享逻辑包
- 后端重构为纯净 Web 服务（无 Tkinter 依赖）

### 新增
- Tailwind CSS 现代化的前端界面
- 前端置信度分布可视化柱状图
- 图片点击放大查看功能
- 键盘快捷键 (Enter) 触发检测

### 改进
- 前端上传区域渐变边框动画
- 卡片式结果展示布局
- 加载骨架屏
- 更好的移动端适配

### 修复
- 重构后所有路径使用绝对路径，避免工作目录问题
- 配置系统支持热加载和热更新
